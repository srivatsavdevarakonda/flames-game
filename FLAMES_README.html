Step-by-step
🔑 How to Get Your Groq API Key
🎉
Yes! Groq is completely FREE No credit card needed. No hidden charges. Free tier is more than enough for this project.
  1. 1
    Go to Groq Console
    Open your browser and go to the Groq developer console. This is where you manage your API keys.
    ↗ console.groq.com
  2. 2
    Sign Up (or Log In)
    Click "Sign Up" — you can use your Google account or create one with email. Takes 30 seconds. No payment info asked.
  3. 3
    Go to API Keys Section
    After logging in, look at the left sidebar → click "API Keys". You'll see your keys dashboard.
  4. 4
    Create a New Key
    Click the "Create API Key" button. Give it any name (e.g. "flames-game"). Click Create.
  5. 5
    Copy the Key Immediately!
    ⚠️ The key is shown only once. Copy it right away. It looks like: gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
  6. 6
    Add it to your .env file
    In your flames_game/ folder, open (or create) the .env file and paste:
    .env 
    # Paste your key here
GROQ_API_KEY=gsk_your_actual_key_here
  7. 7
    Enable AI Story in the App
    Open the FLAMES game in your browser → toggle "Enable AI Story ✨" → calculate — you'll now get AI-generated stories!

Getting Started
🚀 How to Run the Project
Step 1 — Install dependencies (Friend 1)
bash 
# In the backend folder
cd flames_game/backend
pip install -r requirements.txt
Step 2 — Add your Groq key (Friend 2)
.env 
GROQ_API_KEY=gsk_your_key_here
Step 3 — Start the backend (Terminal 1)
bash 
cd flames_game/backend
uvicorn app:app --reload
# → Running on http://localhost:8000
Step 4 — Open the frontend (You)
bash 
# Open frontend/index.html in VS Code
# Right click → Open with Live Server
# → Runs on http://127.0.0.1:5500

⚠️ Keep BOTH terminals running at the same time — backend on 8000, frontend on 5500
